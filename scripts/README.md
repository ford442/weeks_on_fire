# Scripts

Automation and helper tools for the weeks_of_fire production pipeline.

## Available scripts
- `generate-prompts.py` – Turn a synopsis into a batch of Grok Imagine prompts
- `srt-tools.py` – Validate, fix, and time subtitles
- `update-index.py` – Auto-update the cinematic index.html (future)

Run with: `python scripts/generate-prompts.py`

Add new scripts here as the project grows. Keep them lightweight and well-commented.
